# TicketOut
미국내에서 교통법규를 위반하게 되면 미국의 주마다 법이 다르기 때문에 까다로운 상황이 발생하기 쉽습니다.
이 어플리케이션은 미국 교통법규를 더 잘 이해하는데 도움을 주기위한 정보를 공유하기 위해 제작되었습니다.
본 프로젝트의 어플리케이션은 상업적인 용도가 있을 수 있으므로 소스는 공개하지 못하는점 알려드립니다.
